# ARMTemplates
Azure Resource Manager Templates for resource deployments.
The templates shall help you to deploy Azure Machine Learning and supporting services using Private Endpoints and vNet.
Additional services: Azure Bastion, Azure Data Science VM (Windows/Linux), Azure Data Lake Gen2 Storage

# How to deploy:
- Open 01-Solution-Deployment.ps1 file in VSCode or Powershell.
- Update all .parameters.json files.
- Run the 01-Solution-Deployment.ps1 file sections.
